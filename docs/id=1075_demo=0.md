As a potential Bujeti customer,
I book a demo with an expert to learn more about their financial management solutions,
so that I can understand how Bujeti can help simplify my company's financial processes.

When I click on the 'Meet an Expert' link and select a date and time for a demo
Then I should see that the demo date is confirmed for January 6, 2025, at 300 PM, and the booking process should allow me to proceed.
