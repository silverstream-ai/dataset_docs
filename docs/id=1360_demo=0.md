# Task

**User Story for Accessing Educational Resources on Pigeon**

As a user interested in financial education,
I explore educational content about loans and relationships,
so that I can make informed decisions about borrowing and lending money.

**Success definition:** Given I am on the Pigeon homepage
When I click on the Education button and then navigate to 'The Nest' section and select the article 'How Much Does it Cost to Sue Someone? A Comprehensive Guide'
Then I should see the article details and related information displayed.
