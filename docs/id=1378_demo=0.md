# Task

**User Story for Booking an Appointment at LunaJoy**

As a prospective client seeking mental health services,
I book an appointment with a qualified therapist,
so that I can receive the support I need for my mental health in a convenient manner.

**Success definition:** Given I am on the LunaJoy appointment booking page and have reviewed the services available
When I fill in my mobile number in the provided field and click the 'Proceed' button
Then I should be directed to the next step in the appointment booking process, confirming that my input is valid and that I can continue with the appointment scheduling.
