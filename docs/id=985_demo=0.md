# Task

**Early Access Request User Story**

As an interested potential user,
I request early access to the Designbase platform,
so that I can start exploring and using the interface design tool to build and develop ideas efficiently.

**Success definition:** Given I am on the Designbase early access request page
When I fill in my first name, last name, work email and describe my project
Then I should see the 'Request early access' button become enabled and be able to click it to submit my request.
