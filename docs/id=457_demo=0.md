# Task

**Demo Request User Story**

As a prospective customer,
I request a demonstration of Malibou's services,
so that I can evaluate how their payroll and HR solutions fit my company's needs.

**Success definition:** Given I am on the Malibou homepage
When I click on the 'Demander une démo' link, fill out the demo request form with my details, and submit it
Then I should be redirected to a confirmation page indicating that my demo request has been successfully submitted.
