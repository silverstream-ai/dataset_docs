# Task

**User Story - Scheduling a Demo**

As a potential customer in the AEC industry,
I schedule a live demonstration of Cogram's services,
so that I can see how Cogram can help improve my project management and meeting documentation.

**Success definition:** Given I am on the Architecture, Engineering, Construction section of Cogram's website
When I click on the 'Book a Demo' link
Then I should be navigated to a demo scheduling page with a form to fill out my information for a live demonstration.
