    User Story for Account Creation

    As a new user, I create an account using my personal information, so that I can access the services and features provided by Happenstance.

    Success definition: Given I am on the account creation page and have not yet created an account When I fill in the required fields with my personal information and click the 'Continue' button Then I should see a notification confirming that my input is valid and that my account is being created.
