                            StaticText Foursquare just announced they’re giving away their place data for FREE. WHAT? Here’s why that’s a genius move and why I'm excited about what it means for the location industry. Foursquare was one of the only truly global point-of-interest datasets apart from OpenStreetMap. OSM was inferior to Foursquare in coverage, attributes, and recency. I used to buy Foursquare for that reason. Now... you can use that data for free and commercially! What does this mean? 1. This is a huge win for Open Data, Developers and Foursquare. 2. Location data is becoming a commodity to compete with Google. 3. AI makes it much easier to keep track of the world and pushes the value of point-of-interest data to ZERO We'll see this newly released data quickly getting used for Geocoding, Search, and Maps and anyone with spatial data in a data warehouse will start using it without the upfront cost being a barrier to experimenting. So WHY is Foursquare doing this? It's a very ballsy and forward-thinking move. They are building a global operating system where humans and agents can contribute their knowledge of place data. Becoming the place where everyone maintains this commodity will be very valuable. In location data POIs were the last frontier where you couldn't access data for free. Buildings used to be hard to get -> Microsoft building footprints. Addresses used to be hard to get -> OpenAddresses. Validated roads and other data was hard to get -> Overture. But POIs were always hard to get cause maintaining them in OpenStreetMap is near impossible. This also marks now a time where nearly every major company has adopted OpenData as part of their location strategy vs. maintaining a strictly private data asset. In 2014 Mapbox, in 2018 Facebook, Microsoft in 2023 TomTom and in 2024 now Foursquare. Congratulations to Foursquare and I'm excited to contribute and build with it!
